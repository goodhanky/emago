# Emago - Pixel Art Generation Prompts

**Purpose:** AI image generation prompts for all game assets
**Style:** Retro sci-fi space strategy, 16-bit aesthetic
**Last Updated:** 2026-01-05

## Style Guidelines

| Setting | Value |
|---------|-------|
| Pixel Scale | 24px base (icons), 48px (buildings), 64px (ships) |
| Color Palette | Dark blues, purples, metallic grays, accent golds/greens/cyans |
| Format | PNG with transparent background |
| Anti-aliasing | None (crisp pixels) |
